
# Binary Voter Model

A simulation to reproduce the original Binary Voter Model of Clifford and
Sudbury 1973[^1] and Holley and Liggett 1975[^2].

```@meta
CurrentModule = bvm
```

```@docs
run_sim(n=20, p=0.2, make_anim=false, influencer=false, replacement=false)
make_graph(n, p)
```
